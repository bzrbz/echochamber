# echochamber

> Red social cerrada, solo por invitación. Un Reddit de bolsillo para comentar enlaces con amigos.

Un puesto de [bzr.bz](https://bzr.bz). Destino: [echo.bzr.bz](https://echo.bzr.bz).

*(El repo se llama `echochamber` porque ese es el nombre del proyecto. El dominio es `echo.bzr.bz` para dejarlo corto.)*

**Estado:** 🟡 en planificación · MVP pendiente

## Qué es

Un sitio donde un grupo pequeño de amigos (5-50 personas) comparte enlaces y los comenta. Formato tipo Reddit — posts con URL, hilos de comentarios — pero **sin feed público, sin algoritmo, sin descubrimiento, y sin votos**. Solo se entra con invitación de alguien que ya está dentro, y lo que se publica solo lo ven los miembros del círculo.

El nombre es deliberadamente irónico. Una cámara de eco puede ser tóxica cuando es involuntaria y masiva, pero es exactamente lo que uno busca cuando lo que quiere es hablar con gente que comparte sus referencias. Aquí la cámara de eco es *opt-in*, pequeña, y tú sabes quién está dentro.
